1. What is DOM and what is the purpose of DOM?

“ DOM” means document object model. Dom is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

3. What is event bubble?

Event Bubble is a concept in the DOM (Document Object Model). It happens when an element receives an event, and that event bubbles up (or you can say is transmitted or propagated) to its parent and ancestor elements in the DOM tree until it gets to the root element.

4. What is a callback function and why will you use it?

Callback function: A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action. The a example is a synchronous callback, as it is executed immediately.
